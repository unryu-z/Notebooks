## pycharm使用R语言

使用R语言参考https://blog.csdn.net/weixin_44027604/article/details/103368866

其中Anaconda安装R解释器的命令在cmd窗口运行即可：

```
conda install R
```

安装R包命令：

```
conda install -c r r-package_name
```

也可以安装Rstudio（不过可以在pycharm编辑R，可以不按照Rstudio）：

```
conda install -c r rstudio
```
